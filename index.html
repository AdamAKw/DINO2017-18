Grupa CE

Komenda: git config --global user.email "user_name"

Opisz: Ustawia globalnie email użytkownika.

Przykład użycia: git config --global user.email "user@st.amu.edu.pl"

Komenda: git diff --staged

Opisz: Pokazuje co zostało zmienione w obserwowanych plikach.

Przykład użycia: git diff --staged

Komenda: git checkout -b "branch"

Opisz: Przechodzi na gałąź o nazwie branch, jeśli taka nie istnieje to tworzy nową.

Przykład użycia: git checkout -b "user_branch"

Komenda: git log --graph --all

Opisz: Wypisuje listę poprzednich commitów. Opcja graph dodaje graficzny podgląd gałęzi. Opcja all pokazuje też inne gałęzie.

Przykład użycia: git log --graph --all

Komenda: git branch -d "branch"

Opisz: Usuwa gałąź o nazwie branch.

Przykład użycia: git branch -d "user_branch"

Komenda: git fetch "remote" "branch"

Opisz: Pobiera wszystkie zmiany z remote'a o nazwie remote z gałęzi branch.

Przykład użycia: git fetch "remote" "user_branch"
